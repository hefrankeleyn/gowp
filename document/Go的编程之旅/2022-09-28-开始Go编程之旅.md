## 一、构建运行代码示例

```shell
go install golang.org/x/website/tour@latest
```

这将放置一个`tour`二进制到你的`GOPATH`到bin目录下。当你运行这个tour程序时，它将打开一个web浏览器展示你本地tour到版本。
